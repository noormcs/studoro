Module is almost installed, now you need to do a few things: 1. get you google API key for your domain: http://code.google.com/apis/maps/signup.html, and enter it in this module settings. 2. update profiles locations, by running the following script under admin: http://your-domain.com/path-to-dolphin-files/m/map_profiles/update_profiles, but please be patient it can take from several minutes to several days! 3. update cities locations, by running the following script under admin: http://your-domain.com/path-to-dolphin-files/m/map_profiles/update_cities, but please be patient it can take from several minutes to several hours.
